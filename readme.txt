Proyecto final para la materia Gestion de Bases de Datos, el cual consistia en el armado de una base de datos de peliculas, con sus directores, actores y sus generos correspondientes.

Se trabajo con Visual Studio Code, MySQL, DBeaver y Docker.

En el repositorio se encuentra el archivo docker-compose.yml para crear la base de datos en docker, la cual tiene asignada la ruta de la persistencia, tambien se encuentra una imagen modelo.png con el modelo de la base de datos. En el archivo peliculas.sql se encuentra la creacion de las tablas, la insercion de los datos y la resolucion de las consignas del trabajo.
